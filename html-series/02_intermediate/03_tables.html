Creating Tables in HTML

Tables are used to display data in a structured format. Below is an example of how to create a simple table.

Sample Table
Header 1 Header 2 Header 3
Row 1, Cell 1 Row 1, Cell 2 Row 1, Cell 3
Row 2, Cell 1 Row 2, Cell 2 Row 2, Cell 3
Row 3, Cell 1 Row 3, Cell 2 Row 3, Cell 3

Table Attributes

Common attributes for tables include:

  • border: Specifies the border width of the table.
  • cellpadding: Defines the space between the cell content and its border.
  • cellspacing: Sets the space between the cells.

Conclusion

Tables are a powerful way to present data in a clear and organized manner. Use them wisely to enhance the readability of your content.
