HTML Attributes

HTML attributes provide additional information about HTML elements. They are always specified in the opening tag and come in name/value pairs like this: name="value".

Common Attributes

  • id: Used to uniquely identify an element.
  • class: Used to classify elements for styling and scripting.
  • style: Used to apply inline CSS styles to an element.

Using Attributes

Here is an example of how to use attributes in an HTML element:

        <p id="unique" class="text" style="color: blue;">This is a paragraph with attributes.</p>

Conclusion

Understanding and using attributes effectively is crucial for creating well-structured and styled HTML documents.
